Basic information
INS_D

The Ellipse-D is Inertial Navigation Systems (INS) from SBG, providing both orientation and navigation data. The use of GNSS data or other aiding equipment such as odometer or internal barometric sensor can be used to provide accurate navigation data, but also to improve orientation accuracy.
